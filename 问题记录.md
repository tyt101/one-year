#### 尾递归
尾递归优化，是指函数结束调用只能调用自身。这样栈中不会一直push递归函数，尾调用会直接替换函数。<br />代码优化示例如下：
```javascript
function Sum(num) {
  if(num === 1) return 1
  return num + Sum(num - 1)
}

// 尾调用优化
function Sum(num, sum) {
  if(num === 1) return num + sum
  return Sum(num-1,sum + num)
}
```

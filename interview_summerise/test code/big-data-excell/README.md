## Usage

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Run all unit tests

```
npm run test
```

### Run single Test

make sure in `tests -> unit`

```
cd .\tests\unit
```

test your file

```
npx jest [FilePath like .\example.spec.ts]
```

### Format code manually.

```
npm run lint-fix
```

### Use commitizen

```
npx cz
```

### Check dependencies

```
npx depcheck
```
